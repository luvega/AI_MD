# 分子对接与虚拟筛选文献

## 核心条目

- `UOUH33GQ` / `du_dockey_2023`：Dockey，大规模分子对接和虚拟筛选工具。
- `T2O1ECSF` / `agrawal_benchmarking_2019`：蛋白-肽对接方法 benchmark。
- `R2W3SF5S` / `crampon_machine-learning_2022`：机器学习方法在 ligand-protein docking 中的综述入口，用于补充传统 docking score 和 ML 重打分/预测之间的边界。
- `57K986LK` / `gu_benchmarking_2025`：AI-powered docking 在虚拟筛选场景下的 benchmark，用于提醒第三章不要只按单一 score 推进候选。
- `QXKW6K78` / `sui_targeting_2026`：第八章 Advanced Science 正向虚拟筛选范文，展示从代谢依赖 UXS1 到化合物 plantainoside 的靶点-筛选-验证链条。
- `YUMKNHSK` / `shen_structure-based_2026`：第八章 Scientific Reports APE1 虚拟筛选范文，适合作为受体口袋选择、百万级筛选、pose 复核、MD 和 MM/GBSA 解释的案例。
- `Y4ARSYCQ` / `tomarchio_reproducible_2026`：第八章 JCIM 机器学习虚拟筛选范文，强调 scaffold-aware split、applicability domain、ensemble docking、consensus score 和可执行 notebook 的可复现性。

## 项目使用方式

- 第三章方法卡优先引用 `du_dockey_2023`、`crampon_machine-learning_2022` 和 `gu_benchmarking_2025` 支撑小分子 docking/虚拟筛选流程。
- 蛋白-肽 docking、肽链构象和界面复核优先引用 `agrawal_benchmarking_2019`，并在结果表中单独记录肽链构象质量。
- 第八章方法设计可以引用 `sui_targeting_2026`、`shen_structure-based_2026` 和 `tomarchio_reproducible_2026` 作为三类范文：机制驱动靶点筛选、结构口袋驱动筛选、机器学习加 ensemble docking 的可复现筛选。
- 所有 docking/AI docking 文献只作为候选排序和流程设计依据，不把 score 解释为实验亲和力。
