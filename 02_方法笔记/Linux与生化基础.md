# Linux 与生化基础

第一章作为全项目的运行基础，重点用于沉淀命令行、路径、文件格式和生化数据处理常识。

## 本知识库中的作用

- 作为后续对接、分子模拟、Boltz2 解析和 RFdiffusion 资料处理的通用环境基础。
- 后续新增命令、报错和环境配置时，应优先归入本笔记或单独建立环境记录。
- 路径记录要保留 Windows 绝对路径和项目相对路径两种表达。

## 后续补充方向

- 常用 shell 命令清单。
- PDB/CIF/FASTA/SMILES/SDF 等格式速查。
- Windows PowerShell 与 Linux 命令的对应关系。
