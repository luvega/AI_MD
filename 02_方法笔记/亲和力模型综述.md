# 亲和力模型综述

第五章的 `亲和力模型一览表.xlsx` 已按年份、模型/方法、类型、输入、主要输出、强项、局限和当前评价整理了亲和力模型。

## 使用方式

- 做课程讲解时，以表格作为历史线索。
- 做项目选择时，优先按输入类型筛选：序列、结构、复合物、SMILES、蛋白-配体 3D 结构。
- 对 Boltz2 结果解释时，联动 `Boltz2亲和力预测.md` 和 `passaro_boltz-2_2025`。

## 方法选择规则

| 目标 | 优先方法 | 输入要求 | 输出解释 |
|:---|:---|:---|:---|
| 大规模初筛 | docking / 快速 AI 模型 | 受体、配体库、box 或序列 | 只做候选排序 |
| 同系列配体重排序 | MM-GBSA / FEP / AI affinity | 同源结构、可靠配体构象 | 相对比较优先于绝对值 |
| 单个复合物解释 | Boltz2 / AF3 类结构模型 + 可视化复核 | 链、配体、MSA/模板、结构可信度 | 结构合理性和亲和力一起看 |
| 蛋白-肽/PPI | PPI/peptide affinity 工具 + docking/MD | 肽构象、界面接触、复合物结构 | 不能直接套用小分子规则 |
| 机制验证 | MD / FEP / 实验 | 高质量结构和参数 | 成本高但解释力更强 |

## 正式文献锚点

- `passaro_boltz-2_2025` 和 `abramson_accurate_2024`：结构/复合物预测与亲和力解释的主线锚点。
- `wang_deepdtaf_2021`：数据驱动蛋白-配体 affinity 模型示例，适合放在“快速 AI 模型”和“输入模态”讨论中。
- `romero-molina_ppi-affinity_2022`：蛋白-肽/PPI affinity 工具示例，用于强调肽链和蛋白-蛋白界面不能套用小分子 docking 规则。
- `chang_ranking_2023`：AlphaFold 辅助肽 binder 排序示例，用于说明结构预测信号可辅助排序，但不等同实验亲和力。
- `gu_molecular_2023`：MD refinement 与机器学习亲和力预测的边界示例，用于提醒不同靶点上 MD 可能产生正向、中性或负向影响。

## 结果解释字段

| 字段 | 说明 |
|:---|:---|
| `method_family` | docking、MM-GBSA、FEP、AI affinity、Boltz2、QSAR 等 |
| `input_modality` | 序列、结构、SMILES、复合物 3D、轨迹、MSA |
| `score_name` | score、ΔG、pIC50、Kd、probability、confidence 等 |
| `score_direction` | 越大越好、越小越好、越负越好 |
| `comparable_scope` | 仅同批次、同靶点、同系列、跨靶点或不可比较 |
| `confidence_context` | 结构置信度、界面置信度、模型适用域、实验条件 |

## 后续补充

- 标注模型是否支持蛋白-配体、蛋白-蛋白、蛋白-肽或多组分体系。
- 对正式进入项目的模型，补充最小输入、关键输出、适用边界和推荐实验记录模板。
