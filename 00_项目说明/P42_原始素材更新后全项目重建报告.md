# P42 原始素材更新后全项目重建报告

## 本轮目标

本轮按 LLM Wiki 三层规则重新收敛项目：原始素材继续作为只读 source of truth，Markdown wiki 负责解释和连接，`CLAUDE.md` / `AGENTs.md` / scripts 负责约束后续 Agent 行为。

用户确认：原先那版 `book/` 因内容不完整被删除。因此当前 `book/` 只能视为从 `chapters/chapter-XX/正文.md` 重新同步生成的发布层，不反向作为教材正文来源。

## 使用材料

- `06_原始学习素材/`：只读取文件名、扩展名、mtime、PDF 提取报告和 OCR 质量报告，不复制原始 PDF、Office、压缩包或解包内容。
- `大纲.md`、`chapters/chapter-01/本章大纲.md` 至 `chapters/chapter-12/本章大纲.md`、12 个 `正文.md`：作为教材结构和发布层同步来源。
- `CLAUDE.md`、`AGENTs.md`、`index.md`、`log.md`：作为 LLM Wiki 和教材写作约束。
- `tools/sync_online_book.py`、`tools/validate_online_book.py`、`tools/graph_health.py`、`tools/audit_raw_sources.py`：作为重建和验收工具。
