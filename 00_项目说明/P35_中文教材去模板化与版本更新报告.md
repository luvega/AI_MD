# P35 中文教材去模板化与版本更新报告

## Summary

P35 在 P34 可读性增强的基础上继续修订 8 章在线教材正文，目标是消除跨章重复模板句，让每章围绕本章对象、输入、QC、证据边界和下一步任务展开。当前项目版本更新为 `v0.8.0`，在线书籍版本更新为 `v0.8-p35-detemplated`。

本轮不新增科学事实、不新增文献、不新增实验结果，不重做 Imagegen 图，也不改变引用列表、代码块、图片路径、DOI/URL、BibTeX key 或 Zotero item key。

## Key Changes

- 重写第 1-8 章中重复度最高的教学提示段，包括学习目标收束、Imagegen 图注、核心概念说明、方法流程说明、代码案例解释、实验/练习入口、使用边界和延伸阅读。
- `tools/audit_book_readability.py` 的重复句统计改为 prose-only：排除标题、表格、列表、代码块、引用区和图片链接残留。
- 新增 `00_项目说明/book-stage-reports/p35-detemplate-report.md`，记录 P35 改写策略、审计指标和边界保护。
- 更新 `VERSION`、`book/book_map.toml`、在线书籍首页、MkDocs 导航、根索引、操作日志和版本记录。

## Boundary

`06_原始学习素材/` 继续本地保留且不上传。AI_MD 仓库没有保存第三方 skill 副本；外部写作技能只作为写作方法参考。P35 修订后的正文仍是内部教学/组内培训版本，公开发布前还需要版权、截图、图示和人工学术审校。
