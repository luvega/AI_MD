# P38 在线教材报告页迁出报告

## Summary

P38 将 P25、P34、P35、P36 和 P37 的阶段性维护报告从 `book/docs/resources/` 迁出到 `00_项目说明/book-stage-reports/`。这些报告继续作为项目维护和审计材料保存，但不再作为 MkDocs 在线教材内容发布。

当前项目版本更新为 `v0.9.2`，在线书籍版本更新为 `v0.9.2-p38-reports-out-of-book`。

## Key Changes

- 从 `book/mkdocs.yml` 的课程资源导航中移除 P25/P34/P35/P36/P37 报告页。
- 从 `book/docs/index.md` 的课程资源表中移除阶段性报告入口。
- 将阶段性报告集中迁入 `00_项目说明/book-stage-reports/`，并新增该目录索引。
- 更新项目索引、版本记录和相关维护报告中的路径，避免断链。

## Boundary

本轮只调整报告页位置和导航，不修改章节正文、引用、代码、图片、截图、BibTeX key、Zotero item key 或原始素材。`06_原始学习素材/` 继续本地保留且不上传。
