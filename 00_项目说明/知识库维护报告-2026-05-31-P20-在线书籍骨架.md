# 知识库维护报告 2026-05-31 P20 在线书籍骨架

## 本轮目标

把 AI_MD LLM Wiki 生成一套面向课程讲义读者的在线书籍第一版。首版只做可导航骨架：建立 MkDocs Material 工程、8 个章节、5 个附录、章节来源映射和校验工具，不复制原始 PDF、图片或压缩包。

## 新增内容

| 文件或目录 | 说明 |
|:---|:---|
| `book/mkdocs.yml` | MkDocs Material 配置，包含中文站点、搜索、导航和主题设置。 |
| `book/requirements.txt` | 只声明 `mkdocs-material`。 |
| `book/book_map.toml` | 在线书籍章节到 AI_MD wiki 来源、实验模板、研究工作台和 BibTeX key 的 source of truth。 |
| `book/docs/index.md` | 在线书籍首页、章节入口、附录入口和来源边界说明。 |
| `book/docs/chapters/` | 第 1-8 章课程讲义骨架，每章包含 9 个固定区块。 |
| `book/docs/appendices/` | 研究项目池、实验模板、引用表、AI 回归评测和知识边界附录。 |
| `tools/validate_online_book.py` | 检查章节必填区块、来源路径、BibTeX key、内部链接和 raw source 链接。 |
| `tests/test_validate_online_book.py` | 在线书籍校验器的 TDD 测试。 |

## 结构规则

- 在线书籍使用 `book/book_map.toml` 追踪来源，不从原始 PDF 直接生成正文。
- 每个主章节必须包含：本章导读、学习目标、知识图谱入口、核心概念、方法流程、关键文献与 BibTeX key、实验/练习入口、使用边界与常见误读、延伸阅读与下一步。
- 章节正文使用 BibTeX key，不把 Zotero item key 当作正式引用 key。
- 书籍页面不能直接链接或嵌入 `06_原始学习素材/` 中的原始 PDF、图片、压缩包或 Office 文件。
