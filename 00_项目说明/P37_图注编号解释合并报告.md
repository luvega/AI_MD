# P37 图注编号解释合并报告

## Summary

P37 将图下编号解释表并入图注，压缩在线教材版面并集中图示信息。当前项目版本更新为 `v0.9.1`，在线书籍版本更新为 `v0.9.1-p37-caption-compaction`。

本轮只调整图注和版式，不修改图片、不新增科学事实、不新增实验结果，也不改变引用、代码、截图文件、DOI/URL、BibTeX key 或 Zotero item key。

## Key Changes

- 将 8 章知识图谱编号解释并入 `图X.1` 图注。
- 将 8 章流程图编号解释并入 `图X.3` 图注。
- 删除独立编号解释表，使图注成为读者理解图示的集中入口。
- 增强 `tools/audit_book_figures.py` 与 `tests/test_audit_book_figures.py`，防止独立编号表重新出现。
