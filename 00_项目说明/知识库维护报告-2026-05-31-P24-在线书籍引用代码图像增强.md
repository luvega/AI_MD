# P24 在线书籍引用、代码和 Imagegen 图像增强

## 更新范围

- 为 8 个主章节新增 Imagegen 知识图谱和流程解释图，文件位于 `book/docs/assets/imagegen/`。
- 为 8 个主章节新增 `代码案例与软件操作` 区块，包含可复制代码块、配套代码文件、软件/界面截图、操作步骤表和常见错误提示。
- 新增 `book/docs/assets/code/`、`book/docs/assets/screenshots/` 和 `book/docs/resources/` 课程资源层。
- 新增 `book/docs/resources/imagegen-prompts.md`、`imagegen-manifest.tsv`、`code-manifest.tsv`、`screenshot-manifest.tsv` 和 `reproducible-labs.md`。
- 新增 `tools/update_book_references.py`，从 `references/references.bib` 与 `references/zotero-map.tsv` 自动生成 Nature 风格引用区和附录 C。
- 更新 `references/references.bib` 中 `du_dockey_2023` 与 `agrawal_benchmarking_2019` 的 DOI、期刊、卷期和 URL 元数据。

## 内容边界

- Imagegen 图像只作为教学辅助图，不作为精确数据图；精确术语和流程判断写在章节正文表格中。
- 软件截图采用本地 dry-run 教学资产，未复制原始 PDF 图表，也未把第三方网页截图误写成本项目结果。
- 代码案例是 dry-run、配置样例或结果解析脚本，不表示已经完成 GPU 大模型生产运行。
- 第八章补充 PDF 继续作为文献案例和方法借鉴，不能写成本项目运行结果。

## 校验要求

P24 后在线书籍校验应使用增强参数：

```powershell
python tools/validate_online_book.py --map book/book_map.toml --book-root book/docs --min-chapter-chars 5000 --require-nature-refs --require-imagegen
```

引用区可通过以下命令重建：

```powershell
python tools/update_book_references.py --map book/book_map.toml --book-root book/docs --references references/references.bib --zotero-map references/zotero-map.tsv
```
