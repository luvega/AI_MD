# P25 在线书籍正文润色与结构重排

## 更新范围

- 使用 `Research-Paper-Writing-Skills` 的 reverse outline、段落功能和 claim-evidence alignment 原则，重排 8 个主章节正文。
- 使用 `academic-chinese-style` 的中文生物医学写作规则，降级过强表述，统一“提示/支持/表明/仍需验证”等证据边界。
- 新增 `tools/polish_book_chapters.py`，用于受控改写章节正文，并保护引用卡片、代码块、图片链接、DOI/URL、BibTeX key 和 Zotero item key。
- 新增 `book/docs/resources/style-guide.md`，记录在线书籍中文教材正文风格。
- 新增 `00_项目说明/book-stage-reports/polish-report.md`，记录每章 reverse outline、claim-evidence map 和需作者确认项。
