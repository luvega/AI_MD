## v1.0.0 范围

`v1.0.0` 是 12 章正文进入在线教材发布层的版本。发布层从 `大纲.md` 和 `chapters/chapter-01/正文.md` 至 `chapters/chapter-12/正文.md` 生成，不上传 `本章大纲.md`，不复制 `06_原始学习素材/`，也不恢复旧版 `book/docs/` 的阶段报告页。

核心状态：

- 新增 `book/mkdocs.yml`、`book/docs/index.md`、12 个章节页面和蓝白配色 CSS，形成面向 GitHub Pages 的 MkDocs Material 站点。
- 新增 `tools/sync_online_book.py`，从 12 章正文生成发布层，并过滤写作准备区、待作者确认项、旧发布层路径和原始素材路径。
- 新增 `tools/validate_online_book.py` 与 `tests/test_online_book_publish.py`，检查 12 章页面、导航、蓝白主题、链接、禁用来源和大纲泄漏。
- 恢复 `.github/workflows/deploy-book.yml`，在 `master` push 或手动触发时同步、测试、严格构建并部署 GitHub Pages。
- 本轮只整理和发布既有正文，不新增实验结论、模型性能、临床意义或未验证机制。

## v0.9.3 范围

`v0.9.3` 是 P39 之后的历史版本收敛点。P34 在 P33 文献锚点基础上增强在线书籍正文可读性；P35 继续去除跨章模板句；P36 统一 8 章 32 个图位的出版式编号、图下注释、alt text 和来源边界；P37 将图中编号解释合并进图注；P38 将阶段性报告迁出在线教材内容层；P39 使用 `academic-research-suite / academic-paper-reviewer` 审稿框架继续修订正文，补充文献使用说明、案例走读和 P39 审稿就绪审计器。P40 删除旧版发布层，12 章教材生成入口切换为 `大纲.md` 和 `chapters/`。

核心状态：

- 旧版在线书籍曾完成第 1-8 章课程正文、Nature 风格引用、Imagegen 图谱、Mermaid 图示、代码案例和截图入口；这些内容现作为历史记录，不再作为当前 12 章教材正文来源。
- P34 已把 8 章主章节扩写为更适合连续阅读的教材讲解文本，并新增可读性审计脚本和报告。
- P35 已把 8 章重复模板句改写为章节专属表达，prose-only 重复句审计为 0。
- P36 已把每章 Imagegen 知识图谱、Mermaid 结构图、Imagegen 流程图和软件截图统一为 `图X.Y` 图注体系，图注审计为 0 errors。
- P37 已将知识图谱和流程图的编号解释表并入图注，章节内独立编号解释表为 0。
- P38 已将 P25/P34/P35/P36/P37 阶段性维护报告迁出 `book/docs/`，在线教材导航不再展示这些报告页。
- P39 已把 8 章导读、核心概念、方法流程、使用边界和下一步改为更明确的章节专属叙事，并为第 3/5/6/8 章补充案例走读。
- P40 已删除旧版 `book/`、GitHub Pages workflow、旧书专用脚本和旧书测试。
- `references/zotero-map.tsv` 与 `references/references.bib` 已覆盖 29 条正式文献映射。
- P33 已补齐 `5286JS9F` / `chai_discovery_chai-1_2024`、`T2M6L289` / `butcher_novo_2025`、`UIPWC5CR` / `pacesa_bindcraft_2025`。
