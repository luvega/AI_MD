## v1.0.2 范围

`v1.0.2` 是 Wiki / Book 分轨保护版本。它只更新项目 schema、说明文档和本地 skill 规则，不同步 `book/docs`，不运行 `sync_online_book.py`，不改章节正文。

核心状态：

- 新增 `Wiki与Book分轨规则.md`，明确 Wiki 维护默认不更新在线书，Book 写作和发布必须由用户显式触发。
- `CLAUDE.md` 新增 Wiki / Book 分轨硬规则；`AGENTs.md` 明确 `/update-vault`、`wiki-lint`、附件索引、文献映射和方法卡更新不自动触发正文或 `book/` 更新。
- `.claude/skills/update-vault`、`wiki-lint`、`ai-md-router`、`ingest-source` 和 `takenote` 已写入 Book 写入保护。
- Wiki 轨默认验收为 LLM Wiki 校验、raw-source 审计和图谱健康检查；Book 轨才运行 `sync_online_book.py`、在线书校验和 MkDocs build。

## v1.0.1 范围

`v1.0.1` 是原始素材更新后的全项目重建小版本。它不移动、不复制、不上传 raw sources，而是重新审计 `06_原始学习素材/`，补齐根级课程 Markdown 来源索引，修正旧版 `book/` 与当前发布层的边界，并从 12 章 `正文.md` 重新同步 `book/docs`。

核心状态：

- 新增 `tools/audit_raw_sources.py`，用于检查 raw 文件类型分布、PDF 提取覆盖、关键附件索引覆盖和根级 Markdown 来源索引覆盖。
- `06_原始学习素材/_index.md` 与 `05_附件索引/附件清单.md` 已补齐 8 个根级课程 Markdown 原文入口。
- `CLAUDE.md`、`AGENTs.md` 和 `大纲.md` 已明确：旧版内容不完整的 `book/` 已删除；当前 `book/` 只从 `chapters/chapter-XX/正文.md` 同步生成。
- 第 5 章过期来源声明已修正为本机 raw sources 已恢复，但不随在线书发布；页码、截图和具体命令仍需人工复核。
- P42 曾按“全项目重建”执行 `sync_online_book.py`、在线书校验、pytest、MkDocs strict build、LLM Wiki 校验、raw-source 审计和图谱健康检查。P43/v1.0.2 之后，这不再是 Wiki 维护默认流程。

## v1.0.0 范围

`v1.0.0` 是 12 章正文进入在线教材发布层的版本。发布层从 `大纲.md` 和 `chapters/chapter-01/正文.md` 至 `chapters/chapter-12/正文.md` 生成，不上传 `本章大纲.md`，不复制 `06_原始学习素材/`，也不恢复旧版 `book/docs/` 的阶段报告页。

核心状态：

- 新增 `book/mkdocs.yml`、`book/docs/index.md`、12 个章节页面和蓝白配色 CSS，形成面向 GitHub Pages 的 MkDocs Material 站点。
- 新增 `tools/sync_online_book.py`，从 12 章正文生成发布层，并过滤写作准备区、待作者确认项、旧发布层路径和原始素材路径。
- 新增 `tools/validate_online_book.py` 与 `tests/test_online_book_publish.py`，检查 12 章页面、导航、蓝白主题、链接、禁用来源和大纲泄漏。
- 恢复 `.github/workflows/deploy-book.yml`，在 `master` push 或手动触发时同步、测试、严格构建并部署 GitHub Pages。
- 本轮只整理和发布既有正文，不新增实验结论、模型性能、临床意义或未验证机制。

## v0.9.3 范围

`v0.9.3` 是 P39 之后的历史版本收敛点。P34 在 P33 文献锚点基础上增强在线书籍正文可读性；P35 继续去除跨章模板句；P36 统一 8 章 32 个图位的出版式编号、图下注释、alt text 和来源边界；P37 将图中编号解释合并进图注；P38 将阶段性报告迁出在线教材内容层；P39 使用 `academic-research-suite / academic-paper-reviewer` 审稿框架继续修订正文，补充文献使用说明、案例走读和 P39 审稿就绪审计器。P40 删除旧版发布层，12 章教材生成入口切换为 `大纲.md` 和 `chapters/`。
