# P36 图注与图示出版规范化报告

## Summary

P36 针对在线书籍的图示出版规范进行迭代：8 个主章节的 Imagegen 知识图谱、Mermaid 结构图、Imagegen 流程图和软件截图全部采用 `图X.Y` 编号，并在图下方给出可以独立理解图意的详细图注。当前项目版本更新为 `v0.9.0`，在线书籍版本更新为 `v0.9-p36-figure-captions`。

本轮不新增科学事实、不新增文献、不新增实验结果，也不改变引用列表、代码块、图片文件名、DOI/URL、BibTeX key 或 Zotero item key。图注的作用是解释教学图的阅读方式和边界，不提升证据强度。

## Key Changes

- 修订 `book/docs/chapters/chapter-01.md` 至 `chapter-08.md`，为 32 个图位补齐编号、图题、图下注释和 alt text。
- 新增 `00_项目说明/book-stage-reports/p36-figure-caption-report.md`，记录图位体系、边界说明和审计结果。
- 更新 `book/docs/resources/style-guide.md`，新增 P36 图注规范。
- 新增 `tools/audit_book_figures.py` 与 `tests/test_audit_book_figures.py`，将图号、图注和章节外图片纳入自动审计。
- 调整 `book/docs/resources/screenshot-index.md`，资源页只保留截图文件链接，正式图注统一放在章节正文中。
- 更新 `VERSION`、`book/book_map.toml`、在线书籍首页、MkDocs 导航、根索引、操作日志和版本记录。

## Validation Snapshot

正式验收仍以本轮最终命令输出为准。

## Boundary

`06_原始学习素材/` 继续本地保留且不上传。章节图注中的 Imagegen 图、Mermaid 图和 dry-run 截图均为教学辅助材料；它们不替代正文中的参数、QC 标准和文献边界，也不能被写成真实实验结果。
