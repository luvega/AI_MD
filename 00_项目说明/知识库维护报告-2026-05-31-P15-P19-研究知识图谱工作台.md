# 知识库维护报告 2026-05-31 P15-P19 研究知识图谱工作台

## 工具与测试

新增 `tools/graph_health.py`，用于诊断图谱层健康信号：

- managed Markdown 与 frontmatter 页面数。
- 实体表行数。
- typed relations 统计。
- 孤立页面。
- 过期或缺失 `last_reviewed`。
- `references/zotero-map.tsv` 中缺 `references.bib` 条目的 BibTeX key。
- 文献笔记缺 Zotero/BibTeX key。

新增 `tests/test_graph_health.py`，先用临时小 wiki 写失败测试，再实现脚本，覆盖 stale 页面、孤立页、relation 类型、Zotero 行数和缺失 BibTeX key。
