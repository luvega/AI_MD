# 知识图谱实体索引

## 实体类型

| 类型 | 用途 | 命名规则 |
|:---|:---|:---|
| `method` | 方法流程和判断规则 | `method:主题` |
| `target` | 靶点、蛋白、复合物或疾病机制对象 | `target:标准名` |
| `molecule` | 小分子、天然产物、肽或候选配体 | `molecule:标准名` |
| `model/software` | 软件、模型、平台和脚本工具 | `software:名称` |
| `paper` | 已进入 Zotero/BibTeX 的论文或 PDF | `paper:bibtex_key` |
| `experiment-template` | 可复制的实验记录模板 | `template:任务名` |
| `chapter` | 课程章节和章节精读 | `chapter:编号` |

## 查询提示

- 查某个靶点：先搜 `target:*`，再看对应方法、论文和实验模板。
- 查某个方法能否支持结论：先读本页方法实体，再读 [证据与 claims 矩阵](证据与claims矩阵.md)。
- 查某篇 PDF 是否能写入项目结果：先看 `paper:*` 的“当前用途”和“下一步”，P14 文献默认是教学/文献案例，不是本项目运行结果。
