# 附录 E 知识边界与版权说明

本附录说明在线书籍第一版的使用边界。它是内部课程讲义骨架，不是公开教材成品。

## 内容来源边界

| 层 | 在本书中的用途 | 限制 |
|:---|:---|:---|
| 课程资料 | 提供章节结构和教学线索 | 不复制原始 PDF、图片、压缩包或 Office 文件 |
| Wiki 笔记 | 提供章节摘要、方法流程和来源路径 | 正式写作前仍需核对原始资料 |
| 文献证据 | 提供参考文献和方法依据 | 文献结论不能自动外推到本项目 |
| 本项目结果 | 只使用已有实验记录和明确标注的结果 | 没有记录的运行不能写成结果 |
| 研究假设 | 提供下一步实验和项目池 | 不能写成已经证实的结论 |

## 第八章特别边界

第八章补充 PDF 是课程范文和文献案例。它们可以帮助设计研究问题、方法流程和写作结构，但不能被描述为 AI_MD 已经完成的虚拟筛选、PPI 筛选或蛋白设计结果。

## 图表与素材

首版不搬运原 PDF 图表。后续需要图示时，应优先制作原创流程图、实体关系图或实验记录示意图；若使用文献图表，需要单独处理版权和引用。
