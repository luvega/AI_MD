# P35 中文教材去模板化与版本更新报告

## 本轮目标

P35 接续 P34 的可读性增强，重点处理“每章都像同一个模板套出来”的问题。本轮不新增科学事实、不新增文献、不重做 Imagegen 图，也不改变代码文件名、图片路径、DOI/URL、引用列表或 Zotero/BibTeX provenance。修订目标是让 8 章仍保持统一教材结构，但正文句子围绕本章对象、输入、QC、证据边界和下一步任务展开。

P35 的版本目标同步到在线书籍与项目层面：在线书籍版本更新为 `v0.8-p35-detemplated`，项目版本更新为 `v0.8.0`。本轮属于教材正文质量更新，不代表产生新的实验结果。

## 改写策略

- 将 P34 中跨章复用的导读句、核心概念解释、方法流程说明、代码阅读提示、练习入口、使用边界和延伸阅读段落，改成章节专属表达。
- 每章保留相同的教学结构，但每个重点段落必须出现本章具体对象，例如 Linux 环境、结构复核、docking score、轨迹指标、predicted affinity、RFdiffusion/RFD3、AI Agent 或 Chai-1 项目路线。
- 重复句审计改为只统计正文句子，不再把标题、表格行、列表项、代码块、引用区或图片链接残留计入重复句。
- 对第 3/5/6/8 章继续使用保守语气：score、affinity、设计结果、Chai-1 aggregate score 和文献案例只能写成线索、候选、流程参考或待验证项。
